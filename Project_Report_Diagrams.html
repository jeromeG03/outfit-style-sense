Step-by-Step Flow: 1. User enters email on Login page (P1) → Clicks "Forgot Password" → Modal opens with email input field 2. User receives 6-digit code: "847362" → Modal advances to step 2 → User enters code in verification field 6. Frontend (P1) sends POST request: /api/users/verify-reset-code → Payload: {"email": "user@example.com", "token": "847362"} 7. Backend (P5) validates token: → Queries D3 for matching token → Checks: expiry_date > currentTime ✓ → Checks: used == false ✓ → Returns: {"success": true, "message": "Code verified"} 8. Modal advances to step 3 (new password) → User enters new password: "NewSecure123!" 9. Frontend (P1) sends POST request: /api/users/reset-password → Payload: {"email": "user@example.com", "token": "847362", "newPassword": "NewSecure123!"} 10. User redirected to login page → Can now login with new password Security Features: ✓ 6-digit random code (1,000,000 combinations) ✓ 15-minute expiry (prevents replay attacks) ✓ One-time use tokens (marked as used after reset) ✓ Password hashing with BCrypt ✓ Email validation at every step

Match Score Calculation Algorithm

Total Score (0-100 points) = Color Match + Type Compatibility + Pattern Harmony + Season Match 1. Color Match (40 points max) • 16 colors supported: White, Black, Blue, Navy, Red, Maroon, Green, Pink, Yellow, etc. • Complementary color rules: White + {Blue, Black, Navy, Maroon} = 40 points • Example: White Shirt + Blue Jeans = 40/40 ✓ • Example: Red Kurta + Blue Jeans = 0/40 ✗ (not complementary) 2. Clothing Type Compatibility (30 points max) • Top types: Shirt, T-Shirt, Kurta, Blouse, Blazer, Jacket • Bottom types: Jeans, Trousers, Palazzo, Churidar, Leggings, Shorts • Type pairing rules: Shirt → {Jeans, Trousers, Chinos} = 30 points • Example: Shirt + Jeans = 30/30 ✓ • Example: Shirt + Shorts = 0/30 ✗ (not in pairing rules) 3. Pattern Harmony (20 points max) • Pattern types: Solid, Striped, Checked, Floral, Printed • Harmony rules: Solid works with everything • Pattern + Pattern = 0 points (avoid clashing) • Example: Solid Shirt + Solid Jeans = 20/20 ✓ • Example: Striped Shirt + Checked Pants = 0/20 ✗ (both patterned) 4. Season Compatibility (10 points max) • Seasons: Summer, Winter, Monsoon, All Season • All Season items get 10 points always • Same season: Summer + Summer = 10/10 ✓ • Example: Summer Shirt + Summer Shorts = 10/10 ✓ • Example: Summer Shirt + Winter Jacket = 0/10 ✗ Rating Categories: • Excellent Match (80-100 points) → Green badge • Good Match (65-79 points) → Blue badge • Decent Match (50-64 points) → Amber badge • Poor Match (<50 points) → Not shown to user Example Calculation: White Shirt + Blue Jeans Color Match: 40/40 (White + Blue are complementary) Type Match: 30/30 (Shirt + Jeans perfect pairing) Pattern Match: 20/20 (Solid + Solid harmonious) Season Match: 10/10 (All Season + All Season) TOTAL SCORE: 100/100 - EXCELLENT MATCH ✓
