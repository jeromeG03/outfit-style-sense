# Outfit Style sense - Smart Indian Dress Stylist

Outfit Style sense is a world-class, AI-driven fashion recommendation system designed to bridge the gap between traditional Indian heritage and global fashion trends. It provides personalized styling advice, color analysis, and wardrobe management tailored for the modern individual.

## 🚀 Key Features & Modules

The application is structured into six core intelligent modules:

### 1. Global Occasions (`OccasionPage`)
A specialized engine for regional and international celebrations. Whether it's a traditional Indian festival like **Diwali** or **Durga Puja**, or a global **Christmas Gala**, this module provides curated outfit paths for both men and women.

### 2. Tone Harmony (`SkinTonePage`)
Built on color science, this module analyzes skin tones and undertones to recommend the most flattering palettes. It includes:
- Profile-based analysis (Light, Tan, Dark skin tones).
- "Recommended" vs "Colors to Avoid" lists.
- Universal color picks that work for everyone.

### 3. The Trend Bridge (`TrendsPage`)
Aggregates market insights from global fashion hubs like Mumbai, London, and New York. It features:
- A popularity index for various styles (Fusion, Traditional, Minimalist).
- Insights into "Quiet Luxury" and "Saree-Tech" trends.

### 4. Digital Closet (`WardrobePage`)
An intelligent inventory management tool. Users can upload their own items (like a White Shirt or Pink Kurta) to receive:
- Automated matching scores for pants/bottoms.
- Professional stylist feedback on pairings.

### 5. Fusion AI Stylist (`RecommendationPage`)
The flagship module powered by **Google Gemini AI**. Users can provide free-text prompts (e.g., *"I'm attending a London reception but want a touch of Desi glam"*) to receive:
- Detailed outfit descriptions.
- Styling tips and color palettes.
- Cultural significance and global etiquette advice.

### 6. Global Color Psychology (`PsychologyPage`)
An AI-powered tool that analyzes the hidden language of clothing. It interprets how an outfit might be perceived socially across different cultures and the emotional impact it has on the wearer.

## 🛠️ Technology Stack

- **Frontend:** React (ES6+ Modules)
- **Styling:** Tailwind CSS
- **AI Engine:** @google/genai (Gemini 3 Flash/Pro)
- **Routing:** React Router (HashRouter)
- **Typography:** Playfair Display (Serif) & Inter (Sans-serif)

## 🏗️ Architecture

The system follows a modular architecture as visualized in the **System Architecture** section:
1. **User Interaction:** Input through prompts, selections, or closet uploads.
2. **Algorithmic Filtering:** Pre-processing via cultural rules and skin tone matrices.
3. **Generative Synthesis:** Final processing via Gemini AI to create natural language recommendations.

## 🎓 Project Context

This project serves as a comprehensive demonstration of integrating Generative AI into specific cultural domains, specifically Indian ethnic and fusion fashion. It was designed for the Batch of 2025 as a Final Year Project.
